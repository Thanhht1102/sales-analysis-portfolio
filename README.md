# Sales Performance Analysis

## Business Question
- Doanh thu tập trung ở khu vực nào?
- Nhóm khách hàng nào mang lại giá trị cao nhất?



## Dataset
- Nguồn: dữ liệu bán hàng thực tế
- Thời gian: 2024–2025
- Số cột chính: ngày, khách hàng, xã/huyện, doanh thu

---

## Tools
- Excel / Google Sheets
- SQL
- Pivot Table

---

## Data Cleaning
- Chuẩn hóa tên xã/huyện
- Loại bỏ dữ liệu trùng
- Kiểm tra giá trị thiếu

---

## Analysis & Insights
- 20% khách hàng mang lại phần lớn doanh thu
- Một số tuyến đi nhiều nhưng doanh thu thấp

---

## Recommendation
- Tập trung tuyến có doanh thu cao
- Giảm tần suất khách giá trị thấp
